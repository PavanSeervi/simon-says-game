# Simon Says Memory Game

![MIT License](https://img.shields.io/badge/license-MIT-blue)

A modern web-based adaptation of the classic Simon Says memory game. Test your memory by repeating randomized sequences of lights and sounds, with increasing difficulty and score tracking.

## Features
- **Dynamic Sequence Generation**: Randomized patterns increase in length progressively.
- **Audio-Visual Feedback**: Color flashes and distinct sounds for each button press.
- **Score Tracking**: Real-time score display with a high-score leaderboard.
- **Strict Mode**: Optional mode with instant game-over for incorrect inputs.
- **Responsive Design**: Compatible with desktop and mobile devices.
- **Game State Management**: Save/resume progress using browser localStorage.
